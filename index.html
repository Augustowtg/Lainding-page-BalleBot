Balle Bot

Deixe a sua comunidade segura de um jeito fácil

O que ela faz por você?

1. Criar um sistema de verificação capcha para novos membros garantirem que não são robôs

2. Possui um sistema de banimento altomatico para links suspeitos que forem enviados para comunidade

3. Sistema de avisos de banimento manual, para pessoas admistradoras utilizarem
